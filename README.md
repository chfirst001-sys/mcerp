# NardTeo (나드터 - 올인원 관리 시스템)

## 프로젝트 소개
NardTeo(나드터)는 데이터와 업무의 기본 단위인 나드(Nard)들이 유기적으로 모이고 연결되는 공간(터)으로서, 건물 및 다양한 업무를 효율적으로 관리할 수 있는 올인원 웹 애플리케이션입니다.
직관적인 사이드바와 하단 탭 네비게이션을 통해 다양한 건물 관리 모듈에 접근할 수 있습니다.

## 주요 기능
- **건물 관리**: 건물 선택 및 등록
- **회원 관리**: 사용자 및 입주민 관리
- **데이터 관리**: 시스템DB, 로컬DB
- **주요 모듈**: 대시보드, 시설관리, 회계관리, 입주민관리, 광장, 문서 관리

## 사용 기술
- HTML5, CSS3, JavaScript(ES6 Modules)
- Google Material Symbols

## 실행 방법
`index.html` 파일을 Live Server 등 로컬 웹 서버 환경에서 실행하여 확인할 수 있습니다.
