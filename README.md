# MC ERP (MC지기 - 건물 관리 시스템)

## 프로젝트 소개
MC지기는 건물 관리를 효율적으로 지원하기 위해 만들어진 웹 애플리케이션입니다. 
직관적인 사이드바와 하단 탭 네비게이션을 통해 다양한 건물 관리 모듈에 접근할 수 있습니다.

## 주요 기능
- **건물 관리**: 건물 선택 및 등록
- **회원 관리**: 사용자 및 입주민 관리
- **데이터 관리**: DB 관리
- **주요 모듈**: 대시보드, 시설관리, 회계관리, 입주민관리, 광장, 문서 관리

## 사용 기술
- HTML5, CSS3, JavaScript(ES6 Modules)
- Google Material Symbols

## 실행 방법
`index.html` 파일을 Live Server 등 로컬 웹 서버 환경에서 실행하여 확인할 수 있습니다.
